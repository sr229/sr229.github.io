Deno demo (x86_64)

Example of running Deno (x86_64) on browser using container2wasm.

Source Container Image

denoland/deno:alpine-1.43.2

Terminal

[launch deno repl] [inspect(launch "sh")]

It can take some time to load and start the container.

More info including docs and source code is available from container2wasm repo. This page is served from container2wasm-demo-deno repo as GitHub Pages.
